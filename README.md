# Business Performance Dashboard

This project contains an **interactive Sales Dashboard** designed to analyze sales data and performance over time. The dashboard provides key insights into sales by category, state-wise performance, monthly sales trends, customer profitability, and more.

## Contents
- **Dashboard.png**: A screenshot of the sales dashboard.
- **Problem Statement.pdf**: The problem statement outlining the business challenges and project goals.
- **SALESDASHBOARD.xlsx**: The Excel file containing the data and interactive dashboard.
- **salesdata.csv**: The raw sales data in CSV format.

## Features
- **Sales by Category**: Shows sales performance across various product categories.
- **Profit Gained Over Time**: Displays profit trends over multiple years.
- **Sales by States**: Regional sales performance map.
- **Top 5 Customers**: Identifies the top customers by profit.
- **Monthly Sales Trend**: Shows sales performance throughout the year.

## How to Use
1. Open the **SALESDASHBOARD.xlsx** file in Excel.
2. Use the **Slicers** to filter data by year, category, and state.
3. Analyze the interactive charts and visuals for insights into sales performance.
